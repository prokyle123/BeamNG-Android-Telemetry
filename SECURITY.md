# Security and support

Report security issues privately to **auroramediagroup1@gmail.com**.

Do not post any of the following in a public issue:

- DriveLab serial keys
- Offline activation responses
- Payment details or transaction screenshots containing personal information
- License-server admin tokens
- Android signing keys or passwords
- Cloudflare credentials

For ordinary installation, activation, or telemetry support, email the same address with the app version, Android version, a description of the problem, and a screenshot that does not expose sensitive information.
