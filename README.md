# DriveLab Telem

DriveLab Telem is an Android vehicle telemetry and diagnostic application.

## Download

Download the newest signed APK from the GitHub **Releases** section.

## Installation

1. Download the APK onto the Android device.
2. Allow installation from the browser or file manager when Android asks.
3. Open the APK and select **Install**.
4. For future updates, install the newer APK directly over the existing version.

## Update compatibility

All official releases are signed using the same original DriveLab Telem signing key. The original JKS/keystore must be preserved permanently so customers can install updates without uninstalling the application.

## Security verification

Each release includes its APK SHA-256 checksum. Customers can compare the downloaded APK checksum with the checksum shown in the GitHub release notes.

## Support

When requesting support, include:

- DriveLab Telem version
- Android device model
- Android version
- Vehicle year and model
- Adapter or connection method
- A screenshot of any displayed error
